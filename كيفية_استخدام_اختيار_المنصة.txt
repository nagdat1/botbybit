📋 دليل استخدام اختيار المنصة (Bybit & MEXC)

✅ تم إضافة دعم MEXC بنجاح!

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📍 كيفية الوصول لاختيار المنصة:

1️⃣ افتح البوت في Telegram
2️⃣ اضغط على زر "⚙️ الإعدادات"
3️⃣ ستجد في أعلى القائمة زر "🏦 اختيار المنصة (Bybit/MEXC)"
4️⃣ اضغط عليه

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🏦 خيارات المنصات:

📌 Bybit:
   ✅ يدعم Spot (فوري)
   ✅ يدعم Futures (عقود مستقبلية)
   ✅ رافعة مالية متاحة
   ✅ حساب تجريبي متاح

📌 MEXC:
   ✅ يدعم Spot فقط (فوري)
   ❌ لا يدعم Futures
   ❌ لا توجد رافعة مالية
   ⚠️ فقط التداول الفوري

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔗 خطوات ربط API:

بالنسبة لـ Bybit:
1. اضغط على "Bybit" في قائمة المنصات
2. اضغط "🔑 ربط/تحديث Bybit API Keys"
3. أدخل المفاتيح بالصيغة: API_KEY:API_SECRET
4. اضغط "✅ استخدام Bybit"

بالنسبة لـ MEXC:
1. اضغط على "MEXC" في قائمة المنصات
2. اضغط "🔑 ربط/تحديث MEXC API Keys"
3. أدخل المفاتيح بالصيغة: API_KEY:API_SECRET
4. اضغط "✅ استخدام MEXC"

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🔐 الحصول على API Keys:

من Bybit:
1. اذهب إلى https://www.bybit.com/
2. Account → API Management
3. Create New Key
4. فعّل الصلاحيات: Read, Write, Trade

من MEXC:
1. اذهب إلى https://www.mexc.com/
2. Account → API Management
3. Create New API Key
4. فعّل صلاحية: Spot Trading فقط

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📊 اختبار الاتصال:

بعد ربط API:
• اضغط "📊 اختبار الاتصال"
• سيعرض حالة API والرصيد
• إذا نجح الاختبار، يمكنك البدء بالتداول

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️ ملاحظات مهمة:

1. MEXC تدعم Spot فقط - إذا اخترت MEXC سيتم تغيير 
   نوع السوق تلقائياً إلى Spot

2. لا يمكن استخدام الرافعة المالية في MEXC

3. يمكنك التبديل بين المنصات في أي وقت

4. كل منصة تحتاج API Keys خاصة بها

5. يمكنك ربط كلا المنصتين والتبديل بينهما

📝 إرسال الإشارات من TradingView:

في JSON الإشارة أضف حقل exchange:

للتداول على Bybit:

  "action": "buy",
  "symbol": "BTCUSDT",
  "price": "{{close}}",
  "time": "{{time}}",
  "exchange": "BYBIT"
}

للتداول على MEXC:
{
  "action": "buy",
  "symbol": "BTCUSDT",
  "price": "{{close}}",
  "time": "{{time}}",
  "exchange": "MEXC"
}
