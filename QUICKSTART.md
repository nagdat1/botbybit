# ⚡ البدء السريع - Quick Start

## 🚀 خطوات سريعة للتشغيل (5 دقائق)

### 1️⃣ التحميل والتثبيت

```bash
# استنسخ المشروع
git clone https://github.com/nagdat/botbybit.git
cd botbybit

# ثبّت المتطلبات
pip install -r requirements.txt
```

### 2️⃣ الإعداد

انسخ `env_example.txt` إلى `.env`:

```bash
cp env_example.txt .env
```

عدّل `.env`:

```env
TELEGRAM_TOKEN=YOUR_BOT_TOKEN
ADMIN_USER_ID=YOUR_USER_ID
```

### 3️⃣ التشغيل

```bash
python main.py
```

### 4️⃣ الاستخدام

افتح البوت في Telegram وأرسل `/start`

## 🎯 الخطوات التفصيلية

### A. الحصول على Telegram Bot Token

1. افتح [@BotFather](https://t.me/BotFather)
2. أرسل: `/newbot`
3. اتبع التعليمات
4. انسخ التوكن

### B. معرفة User ID الخاص بك

1. افتح [@userinfobot](https://t.me/userinfobot)
2. سيرسل ID الخاص بك
3. ضعه في `ADMIN_USER_ID`

### C. أول صفقة تجريبية

```
1. في البوت، اختر "حساب تجريبي"
2. اضغط "التداول"
3. اختر "🟢 شراء"
4. اختر "📊 Spot"
5. اختر "BTC/USDT"
6. أدخل المبلغ: 100
7. أكّد الصفقة
```

## 💡 نصائح سريعة

### ✅ افعل
- ابدأ بالحساب التجريبي
- استخدم Stop Loss دائماً
- ابدأ بمبالغ صغيرة
- تعلم قبل التداول الحقيقي

### ❌ لا تفعل
- لا تستخدم رافعة عالية في البداية
- لا تتداول بأموال لا تستطيع خسارتها
- لا تدخل صفقات بدون خطة
- لا تتبع العواطف

## 🎓 أمثلة عملية

### مثال 1: صفقة شراء بسيطة
```
النوع: Spot
الزوج: BTC/USDT
الاتجاه: BUY
السعر: $50,000
المبلغ: $500
الكمية: 0.01 BTC

الهدف: $52,000 (+4%)
Stop Loss: $49,000 (-2%)

الربح المتوقع: $20
الخسارة القصوى: $10
```

### مثال 2: صفقة Futures مع رافعة
```
النوع: Futures
الزوج: ETH/USDT
الاتجاه: BUY
السعر: $3,000
الرافعة: 5x
المبلغ: $200
القوة الشرائية: $1,000
الكمية: 0.333 ETH

الهدف: $3,150 (+5%)
Stop Loss: $2,950 (-1.67%)

الربح المتوقع: $250 (125% من الهامش)
الخسارة القصوى: $83.5 (41.75% من الهامش)
```

### مثال 3: استراتيجية الإغلاق الجزئي
```
الدخول: $1,000 في BTC عند $50,000

الخطة:
عند +3%  → أغلق 25% (ربح $7.5)
عند +5%  → أغلق 25% (ربح $12.5)
عند +8%  → أغلق 25% (ربح $20)
عند +15% → أغلق 25% (ربح $37.5)

إجمالي مؤمن: $77.5
المخاطرة المتبقية: $0
```

## 🆘 حل سريع للمشاكل

### البوت لا يستجيب؟
```bash
# أعد التشغيل
Ctrl+C
python main.py
```

### خطأ في API؟
```
1. تحقق من المفاتيح في Bybit
2. تأكد من الصلاحيات
3. أعد إدخال API:
   /setapi NEW_KEY NEW_SECRET
```

### رصيد الحساب التجريبي انتهى؟
```
الإعدادات → الحساب → تجريبي
سيعيد الرصيد إلى $10,000
```
